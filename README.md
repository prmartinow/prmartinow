### What I'm Building

*   **[repo-analysis-workbench](https://github.com/prmartinow/repo-analysis-workbench)**: Helping your agent understand code repos faster.

### Focus Areas

*   **Context Engineering:** Pruning code trees, mapping repository structures, and optimizing input tokens for local models.
*   **Local Inference Pipelines:** Deploying and benchmarking embedding/reranking APIs (like Qwen) and local OCR systems (PaddleOCR).
*   **Agent Harness & Tooling Evaluation:** Unifying local execution environments for diverse agent runtimes and evaluating agent tooling.
*   **Data Ingestion & Sinks:** Streaming agent telemetry using ClickHouse and Redpanda.
